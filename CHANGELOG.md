# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-01-XX

### Adicionado

- Estrutura inicial do projeto React Native com Expo
- Sistema de tema centralizado (cores, tipografia, espaçamento)
- Tela Home com header, countdown, informações do evento e publicações
- Tela Playlist com lista de músicas
- Navegação customizada entre telas
- Componentes reutilizáveis:
  - BottomNavigation
  - Countdown
  - InfoCard
  - PostCard
  - TrackItem
  - SectionTitle
  - ScreenContainer
  - Logo (SVG)
- Integração de SVG com react-native-svg
- Configuração do Metro Bundler para processar SVG
- TypeScript com tipagem completa

### Estrutura

- Organização modular de componentes
- Separação de responsabilidades
- Sistema de temas profissional
- Componentes globais reutilizáveis
