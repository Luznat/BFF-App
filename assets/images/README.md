# Assets - Imagens

Coloque aqui as imagens do projeto.

## Logo do App

Coloque o arquivo da logo do app com o nome `logo.png` nesta pasta.

**Recomendações:**

- Formato: PNG com fundo transparente
- Tamanho recomendado: 240x80px ou proporção similar
- Resolução: @2x ou @3x para diferentes densidades de tela

## Estrutura

```
assets/images/
├── logo.png          # Logo principal do app
└── README.md         # Este arquivo
```
